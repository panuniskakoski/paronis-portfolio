Just for fun

Now these here are highlights of my random doodles and misc art projects. Everything is mostly made just for the fun of it. They are not in any particular order. Oldest are most likely done about 5 years ago.

House visual proto

Doodle of a game prototype. I was mainly testing the visual concept and lighting. This was done very quickly and there are few visual glitches to be fixed. All designs, assets, riggs and key frame animations are made by me (excluding wall / floor textures). I plan on iterating this concept in the future and have some neat game mechanics planned for it. Game engine is Unity.


Family rigged

I designed, rigged and animated these characters as a 2D rigging exercise. These guys could fit for a turn based RPG maybe?


Test_Guy Rigging Demo

I did this as a 2D rigging exercise. The base puppet has been designed so that it is easy to switch his gear. All assets, riggs and key frame animations are made by me. Game engine is Unity.
