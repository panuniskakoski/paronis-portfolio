Cube VR

About the project

Completely solo project (2021). Project took around 45h of work. All assets, audio and programming is my work. Learned a lot about VR and 3D environment while doing. This was my first 3D game ever.

This is a escape room styled puzzle game, where you have to put the right cube on the pedestal in front of you. Look around you and inspect the environment to find clues. Only tested on Oculus CV1 and Oculus Quest. Game files contain a hint sheet which includes about 15 hints for the puzzles.

The best way to see all the work I've done on the game is to simply play it in VR. You can't see all the little particles and such on the screen. Download the apk from my Google Drive. Link below. I added also link to the projects GitHub repository.
