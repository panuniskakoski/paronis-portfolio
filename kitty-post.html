Kitty Could't Stop

About the project

This is an Android demo I did recently. Took me around 30h to finish. I did all the assets, animations, programming and music (drum track by Jaro Väisänen). Sound effects were downloaded from Freesound, but I did some editing with them.

In this simple mobile game you play as a kitty trapped in an helicopter that can't stop. How far can you survive?

The best way to see all the work I've done on the game is to simply play it. You can download the game for Android from my Google Drive. Link below.
