About the project

Completely solo project (2020). I can't recall exactly, but this project took around 80h of work(??). All assets and programming is my work. This was my first Unity project, so there was a lot of learning involved.

Our task was to build a prototype around "demake" theme. I intented to imitate old school RuneScape, but the idea got a little out of hand along the way.Idea is that there are three tiers in the game wich you gain when you fill each skill to the max (aka Runescape grind, just a lot quicker). On each tier the games graphics get better as I remake the graphics based on the earlier ones. Hence Dev_(ev)olution as I also get better. Derp.

I was mostly focusing on the frame by frame animations. As this was also my very first Unity project, there was a lot to learn.

The game is downloadable from my Google Drive (PC). You can also find the game project repository from my GitHub page.
